Monthly Payment Formula

P = A / D

P = Monthly Payment
A = Loan Amount
D = Discount Factor

D = {[(1 + i) ^n] - 1} / [i(1 + i)^n]

n = Number of Periodic Payments
i = Periodic Interest Rate (Annual rate divided by number of payments per years)

Source: http://banking.about.com/od/loans/a/loan_payment_calculations.htm

Payment Plan Formula

This month interest = ((Interest rate/100) / 12) * Last month balance
This month principal = Monthly Payment - This month interest
This month Balance = Last month balance - This month principal
